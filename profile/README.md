## Featured Projects

### 🛰️ IRONSIGHT

<a href="https://github.com/NoblerWorks-HQ/IRONSIGHT"><img src="https://raw.githubusercontent.com/NoblerWorks-HQ/IRONSIGHT/main/Ironsight.png?v=2" alt="IRONSIGHT Dashboard"></a>

Real-time OSINT command center for monitoring the Middle East conflict. Aggregates open-source intelligence from **50+ sources** — news, Telegram, military tracking, financial markets, and more — into a single live dashboard. No API keys required. Completely free to run.

Live intel feed, auto-translated Telegram OSINT, an interactive theater map with aircraft/naval/strike tracking and missile trajectories, live Israel missile alerts, prediction markets, and defense/energy/crypto market data.

### 🌍 War Games

<a href="https://github.com/NoblerWorks-HQ/WarGames"><img src="https://raw.githubusercontent.com/NoblerWorks-HQ/WarGames/main/screenshots/wargames-nuclear-launch-detected.png" alt="War Games — Nuclear Launch Detected"></a>

AI-powered geopolitical strategy simulation. Three great powers — **NATO, Russia, and China** — compete for territory, resources, and global dominance on an interactive map. Each faction is driven by an AI making strategic decisions every turn. Mix and match providers — let Claude command NATO while Gemini runs Russia and ChatGPT leads China.
